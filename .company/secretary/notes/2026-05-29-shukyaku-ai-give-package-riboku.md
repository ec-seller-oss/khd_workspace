# 集患AI GIVEパッケージ（医者向け）

## 掴みの一言（営業色ゼロ）
> 「先生、口コミ返信とかインスタの集患、AIに丸投げできるの知ってます？ 5秒で出ます。
>  プロンプトお渡しするんで、よかったら今やってみましょうか」
→ KHD/不動産/コンサルの匂いは一切出さない。「便利をGIVE」だけ＝野口型の起点。

## 配布プロンプト3点（コピペ完成形）

### ① Google口コミ返信AI
```
あなたはクリニックの院長です。以下の患者口コミに、①感謝 ②具体的内容への言及
③安心・再来院につながる一言、を含む温かい返信を120字前後で作成してください。
医療広告ガイドラインに配慮し、治療効果の保証・誇大表現・他院比較は避けること。
【口コミ】＜ここに貼る＞
```
※先生の口調に合わせたい場合は、末尾に「〜な口調で（例：丁寧で柔らかい／簡潔で誠実な）」を追加。

### ② SNS集患 台本AI（インスタ/YouTube Shorts）
```
あなたは集患に強い医療系SNSプロデューサーです。【診療科：＜例）内科＞】の患者が
よく抱く不安・疑問を5つ挙げ、各テーマを60秒ショート動画の台本に落としてください。
構成は フック(10秒)→本編→CTA。専門用語は中学生でも分かる言葉に言い換え。
医療広告ガイドライン順守、断定的な効果表現は禁止。
```

### ③ 患者向け説明文・院内掲示AI
```
あなたは患者対応に長けたクリニック広報です。以下の医療内容を、患者が不安なく
理解できるやさしい日本語の説明文（HP・院内掲示用、300字）にしてください。
専門用語には（　）で平易な補足を付けること。
【内容】＜ここに貼る＞
```

## 差別化の核
3本とも「**医療広告ガイドライン配慮**」を明記。医者が集患で最も怖いのが行政指導。
「AIが勝手にガイドラインを守る」が刺さる＝ただのChatGPTお遊びと一線を画す。

## 飲み会での見せ方（最小実験）
1. その場で先生の実口コミ1件をコピペ→①を実演（5秒で「おお」）
2. 「プロンプト集お渡しします」とNotion/PDF共有→継続接点（月1 Tips配信）の名簿化を自然に
3. 営業はしない。後日「外で個別に」が出口（②経営数字AI→開業/承継の会話）

## 検証ログ（2026-05-29 李牧QA・実テスト合格）
3本ともサンプル入力で実行し、出力品質＆医療広告ガイドライン順守を確認＝**実戦投入可**。
- ① 混在クチコミ（待ち時間30分＋説明は丁寧）→ 謝罪→感謝→改善→再来院の型で着地。効果保証/比較なし＝OK。口調指定の注記を追加し確定。
- ② 内科で不安テーマ5本＋完成台本（降圧薬）→ 断定回避・自己判断中止の戒め込みでガイドラインOK。フックで離脱防止。
- ③ インフル予防接種の掲示文→「完全に防げるわけではない/期待できます」で断定回避、専門用語なし、300字弱。掲示そのまま可。

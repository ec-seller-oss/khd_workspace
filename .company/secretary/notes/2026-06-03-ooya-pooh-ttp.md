## 1. プーさんの「稼ぐ構造」（判明分）

```
攻め（フロー）  : 売買利益10億超/年 ← これが本体エンジン
守り（ストック）: 家賃9,000万・CF1,000万 ← BSの土台
方針転換       : 33歳CF4,000万 → 38歳CF1,000万（あえて返済に振りBS強化）
```

主力手法 = **OC（オーナーチェンジ＝入居者付き・割安）で買う → 借入比率を高く取る → 退去後に実需（空室・割高）で売る**
　＝ **OC相場 ⇔ 実需相場の価格裁定（アービトラージ）を取る**のが10億エンジンの正体
＋ 地方は **高利回り現金買い**（無借金の含み資産で純資産を支える／OC裁定とは別ライン）

### OC⇔実需 裁定の仕組み（受講①＋菊池補足で確定）
```
①OC物件を割安で仕入れ … 入居者付き＝住宅ローン不可＝買い手が投資家に限られ「安い」
                        収益物件評価で融資が付く＝借入比率を高く取れる＝自己資金を抑える
②保有して家賃CFで返済を回す … この間に元金を厚く返してBS強化（=CF4000万→1000万の正体）
③入居者が退去 → 空室化 … 実需（マイホーム客）が住宅ローンで買える状態に
④実需向けに高く売る … 実需相場は投資相場より2〜3割高い → 価格差がキャピタル粗利
```

## 2. TTPフレーム ― 盗むべき5要素と「受講で聞き出す空欄」

| # | 盗む要素 | 判明分 | 🟡受講で埋める空欄（質問リスト） |
|---|---|---|---|
| ① | **仕入ルート** | 過去取引先から仕入再開 | どこで・誰から・どう関係を再構築したか。業者/エンド/任売どの比率か |
| ② | **OC物件の見極め＋収益融資** | OCを借入比率高く買う | どんなOC物件を狙うか（利回り/入居者属性/退去見込み）。どの銀行が収益評価で何%まで貸すか |
| ③ | **退去を出口に変える設計** | 退去後に実需で売る | 退去のタイミング/促し方（立退き・更新拒否の可否）。空室化→実需売りまでの保有期間 |
| ④ | **実需出口の販路** | 実需に高値で売る | 販路（仲介付け方/レインズ/自社客）。実需相場に乗せる加工度（軽リフォーム範囲） |
| ⑤ | **地方現金買い基準** | 高利回り・無借金 | 利回り何%以上で買うか。土地値・出口（賃貸/転売）の置き方 |

→ **次回受講のミッション：①②を最優先で聞き出す**（KHD最大の壁＝仕入資金を突破する核）

## 3. KHD玉川式との突合

| 指標 | 玉川式・辛口基準 | プーさん手法の当てはまり |
|---|---|---|
| 仕入利回り | 参考（年間賃料÷仕入） | **OC＝入居者付き＝買った瞬間から利回りが付く**。インカム基準で仕入れられる |
| CCR 15〜20% | 自己資金リターン | OCは収益評価で**借入比率を高く取れる→自己資金を抑える**＝CCRが跳ねる（自己資金ゼロではない） |
| 粗利率 20%以上 | キャピタル（出口） | **実需売り＝投資相場より2〜3割高い**。OC相場⇔実需相場の差がそのまま粗利 |
| CF率 1.5〜2.0% | インカム（保有中） | 保有期間は家賃でCFを取りつつ、ここを"あえて返済に回して"BS強化＝CF4000→1000の正体 |
| 土地値割合 0.4以上 | 担保価値 | 地方現金買いライン＝土地値高い物件に合致（高松144%が実例 [[project_takamatsu_iwate]]） |

**示唆**：OC裁定は**インカム（CF率）とキャピタル（粗利率）を1物件で両取り**する手法＝KHDが既に持つ評価軸が両方そのまま使える。さらにOCは収益評価で借入比率を高く取れるため、最大ボトルネック「**仕入資金不足**」（CFO指摘 [[project_hiring_decision]]）を自己資金を抑えて突破できる。

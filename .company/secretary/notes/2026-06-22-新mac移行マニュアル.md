## 実機調査(2026-06-22)で確定した移行対象
- Claude Code 2.1.158 / Node v24.14.1 / npm 11.13.0（prefix `~/.npm-global`・cache `~/.npm-cache-khd`）
- uv/uvx 0.11.14、Python3(9.6)＋pptx/openpyxl/pandas/google系
- MCP：notion(stdio npx)・googletasks(http Composio)・tradingview(.mcp.json uvx)＋Drive/Gmail/Calendar(アカウント連携)
- 自作CLI：`~/.local/bin`（khd-log/khd-scan/khd-yt-upload）
- 設定一式：`~/.claude/`（CLAUDE.md/agents/skills/projects/memory/scheduled-tasks/plugins）・`~/.claude.json`(MCP+トークン)・`~/.zshrc`
- プロジェクト：`~/01_honbu_docs_automation/`（.company秘書室・scripts含む）
- Homebrew/gh は未導入（移行不要）
- 旧Mac・新Macとも Apple Silicon arm64 → 移行は素直

## 結論（マニュアルの骨子）
- 最短＝**移行アシスタント（手順A）＋再ログイン3点（Claude/Drive/Apple ID）**で9割完了。

- ⚠ トークン/URLは資料に書かない（旧.claude.json or 1Passwordから移す）。

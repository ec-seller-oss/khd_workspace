# 先生＋医療事務 共通プレゼン台本（クリニック全体が良くなる）

## 大原則（中核信条）
売り込まない。**「皆さんが楽になる仕組み」**をGIVEとして見せる。効果は一緒に測る（押し付けでなく検証）。合わなければやめられる、を最初に伝える。

## 0. まず「それぞれの見たい点」を押さえる（プレゼン設計）
| | 医療事務が見たい点（日々の負担） | 先生が見たい点（経営・自分） |
|---|---|---|
| 電話 | 鳴り止まない電話・クレーム対応が減る | 人件費・離職リスクの低減 |
| 受付 | 聞き取り・紙→電カル転記の二度手間が消える | 患者満足（待ち時間・連絡） |
| 待ち時間 | 順番クレームが減る | 院全体の回転・評判 |
| 月末 | まとめ会計の残業が減る | 残業代・労務リスク |
| 自分 | 「私たちが楽になる」実感 | 先生の書類・連絡の手間も減る |
→ **両方に「あなたの困りごとが、ここで減る」と名指しで触れる**のがコツ。どちらかだけの話にしない。

## 1. 冒頭（共通・30秒）
「このLINE、患者さん・先生・スタッフの**三方が楽になる**仕組みです。今日は実際に動くものをお見せします。アンケート（事務・看護師の声）を読み込んで、**皆さんの困りごとに合わせて**作りました。」

## 2. 動く実物を見せる（スマホで・1〜2分）
- メニューをタップ → その場で自動応答が返る
- 「予約」「診療時間」「持ち物」を打つ → 自動で回答
- Web問診フォームを開く → 来院前に記入できる
- 「頭が痛い」など → ちゃんと「お電話・受付で」に振り分く（医療判断は人へ）
→ **「掲示やマニュアルでなく、患者のスマホに届く」**のが肝。

## 3. 【医療事務パート】あなたたちが、ここで楽になる（2分）
- 鳴り止まない電話 → **FAQ自動応答が一次対応**。難しい質問だけ電話に絞られる。
- 受付の聞き取り・転記 → **Web問診で来院前に取得**。紙に書いて電カルに写す二度手間が消える。
- 待ち時間クレーム → **順番をLINEで自動通知**（「あと◯人」）。
- 月末のまとめ会計の残業 → **CPAP等の定期受診リマインド**で通院を平準化。
→ 「**日々の"これ嫌だな"が、具体的にここで減ります**」と1つずつ。

## 4. 【先生パート】経営とご自身（2分）
- 残業・人件費・離職リスクの低減（スタッフが定着しやすい環境）
- 患者満足（待ち時間の見える化・連絡のスムーズさ）
- **先生ご自身の書類・連絡の手間**も減る
- そして **数字で見える**（次の効果測定へ）

## 5. 効果測定＝一緒に測る（KPI・1分）
「導入の前後で、**こちらが測ります**。例えば 受付の電話本数 / Web問診の記入率 / 受付の転記時間 / 順番クレーム / 定期受診の来院率 / 月末残業。**導入前にベースラインを取って、1〜2ヶ月で比較**します。」
→ **「どの指標を見るかは、医療事務の皆さんと相談して決めます」**（現場が無理なく測れる粒度に）。押し付けでなく公平・透明に。

## 6. クロージング（小さく始める・30秒）
「全部を一度にやりません。**まず1つだけ無料で試しましょう**（再診リマインド or FAQ）。効果が出た所から広げます。合わなければやめられます。」
→ 中核信条：効果が出た"分だけ"。ロックインなし。

## 当日の持ち物・段取り
- スマホ（動くLINEを実演）／パイプライン4枚スライド／このKPI表
- 医療事務には「今の電話本数・残業、ざっくり何件/何時間ですか？」とその場でベースラインを軽くヒアリング（測定の入口）。
- ⚠️ 固有値・KPIは現地ヒアリングで変わる前提。決め打ちしない。

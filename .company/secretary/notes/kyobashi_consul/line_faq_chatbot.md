# 京橋 FAQ自動応答チャットボット（実装）

## 役割（宮崎指摘の実体化）
問い合わせの大半をLINEが即・自動で回答（電話ゼロ）。医療判断・クレーム・専門証明など難しい質問**だけ**「お電話はこちら」へエスカレーション → 受付の電話本数が減る。

## ⚠ 医療安全の絶対ルール（チャットボットに必須）
- **症状・診断・治療・お薬の可否など医療判断は絶対に自動回答しない** → 必ず電話/受付へエスカレーション。
- 自動応答は「事務的な案内（時間・予約・持ち物・アクセス・ルール）」に限定。

## A. 自動回答ルール（キーワード → 回答）※固有値は要確認
| # | 反応キーワード例 | 自動回答（コピペ可） |
|---|---|---|
| 1 | 時間/何時/診療/やってる/営業 | 診療時間は平日 9:00-12:30 / 15:00-18:30 です（※要確認）。休診日はメニュー「診療時間・休診」をご覧ください。 |
| 2 | 順番/待ち/アイコール/WEB受付 | WEB受付の方は受付時間内にご来院ください。順番が近づくとLINEで「あと3人です」と自動でお知らせします。 |
| 3 | 予約/変更/キャンセル | ご予約はメニュー「ご予約・再診」から。変更・キャンセルは「予約変更」とお送りください。CPAPなど定期の方は登録で毎月リマインドします。 |
| 4 | 保険証/マイナ/本人確認 | マイナ保険証または保険証をお持ちください（※運用は要確認）。ご不明点は受付でご案内します。 |
| 5 | 持ち物/必要なもの | 保険証・医療証・お薬手帳、紹介状（お持ちの方）をご用意ください。 |
| 6 | 駐車/アクセス/場所/地図/最寄 | アクセス・地図はメニュー「アクセス・電話」から。最寄り駅からの行き方もご案内します。 |
| 7 | CPAP/シーパップ/睡眠 | CPAPの方は毎月の定期診察をお願いします。ご予約は「ご予約・再診」から。データ確認と処方継続のためご来院ください。 |
| 8 | 問診/Web問診/事前 | 来院前にスマホでWeb問診を記入できます。当日の待ち時間が短くなります（メニュー「Web問診」）。 |
| 9 | 休診/休み/祝日/年末 | 休診日はメニュー「診療時間・休診」のカレンダーでご確認ください。 |
| 10 | 化学物質/過敏症（一般案内） | 一般的なご案内はFAQに掲載しています。個別のご相談・証明書などは専門的判断が必要なため、お電話または受付でお願いします。 |

## B. エスカレーション（難しい質問 → 電話案内）
- **反応キーワード**：症状/痛い/痛み/熱/つらい/具合/診断/治る/副作用/クレーム/苦情/証明書/診断書、およびA・Bどれにも一致しない質問。
- **返す文**：
```
お問い合わせありがとうございます。こちらの内容は、お電話または受付でご案内します。
☎ 03-____-____（受付時間：平日 9:00-18:30 ※要確認）
お急ぎの場合はお電話ください。
```

## C. エルメ/LINE設定（流し込み）
1. LINE公式の「応答設定」＝**チャット＋自動応答 両方ON**。エルメ運用なら Webhook をエルメへ。
2. 上表A＝エルメ「キーワード応答」or LINE「応答メッセージ（キーワード一致）」に1行ずつ登録。
3. B＝「一致しない時のデフォルト応答」に電話案内文を設定（フォールバック）。
4. リッチメニュー「よくある質問」タップ→ FAQカルーセル（A の主要6件）を表示。

## D. 要確認リスト（クリニックに確認＝仮置き中）
- 診療時間・休診日・昼休み区分／電話番号／住所・最寄り・駐車場／マイナ保険証の運用／CPAP定期の案内可否／化学物質過敏症の対応方針（証明書の可否）。

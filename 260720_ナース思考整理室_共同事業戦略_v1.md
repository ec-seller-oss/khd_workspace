## 2. 競合マップと比較分析

| プレイヤー | 型 | 価格帯 | 強み | 弱み（＝付け入る隙） |
|-----------|----|--------|------|---------------------|
| 日本看護協会 メンタルヘルス相談窓口 | 公的・電話相談 | 無料 | 信頼性・匿名性 | 単発対応。継続伴走なし |
| 日本看護コーチ協会 | コーチ育成研修（B2B/資格） | 数万〜十数万円/コース | 体系化された研修 | 現場ナース個人の悩みに直接寄り添う場ではない |
| マジキャリ等 キャリアコーチング | B2C高額コーチング | 数十万円 | 体系的プログラム | 看護特化でない・高額・転職前提になりがち |
| NPO法人 看護職キャリアサポート | 看護師経験者のカウンセリング | 約1万円/時間 | 当事者性 | 単発型・デジタル展開弱い |
| posiNurse | 看護師専門パーソナルコーチング | 月額制 | 看護特化 | 学習・キャリア寄り。メンタル×思考整理は主軸でない |
| 転職エージェント（レバウェル看護等） | 人材紹介 | 求職者無料 | 集客力・資本力 | **転職成立が収益源＝中立でない**。「辞めない支援」は提供不可 |
| 外部EAP事業者（M3ヘルスデザイン等） | B2B法人契約 | 月額固定＋従量 | 法人向け実績 | 汎用EAPで「看護師の現場文脈」への解像度が低い |

### cocoro_rescue の差別化ポジション（仮説）
1. **当事者性**: 現役（と推定される）男性看護師本人が運営 → 協会・企業には出せない一人称の言葉。
2. **非・転職誘導**: 「答えを教えない」コーチングスタンス → エージェントと真逆の中立性。
3. **男性看護師という希少性**: 男性看護師は全体の約1割未満。同性のロールモデル不足という固有ペインにも刺さる。
4. 弱み: 個人発信のみでは集客・商品化・法人営業・システムがボトルネック。**← ここがKHDの出番**。

## 4. 共同事業シナリオ（3フェーズ）

### Phase 0: 検証・業務提携（0〜3ヶ月）— JVはまだ作らない
- まず**友人として連絡し、記事の意図と本気度を確認**（最重要。事業の話はその後）。
- 合意できたら軽い業務提携（覚書レベル）:
  - KHD: note/Instagram/LINE のファネル設計・制作を無償〜低額で支援
  - 対価: KHD経由売上のレベニューシェア 20〜30%
- 商品: 60分オンライン思考整理セッション 8,000〜12,000円（NPO相場1万円/時に整合）
- KPI: noteフォロワー、LINE登録数、初回セッション数、リピート率、単価受容性
- **撤退基準を先に決める**（例: 3ヶ月でセッション月10件未満なら提携縮小）

### Phase 1: B2C商品化（3〜9ヶ月）
- 月額メンバーシップ（note/LINE、980〜2,980円）: 週1音声・月1グループセッション・匿名相談
- パッケージ商品: 「新人ナースの1年目伴走」「プリセプター期サポート」等の期間型
- セミナー: KHDのスライド量産体制で月1オンラインセミナー → バックエンドへ誘導
- 目標ライン例: 会員300人 × 1,980円 ≒ 月60万円 + 個人セッション

### Phase 2: B2B2C本命 — 「看護師定着支援プログラム」（9〜18ヶ月）
- KHDの医療機関コンサル顧客へ、**離職防止パッケージ**として法人販売:
  - 集合研修（思考整理メソッド）+ 希望者1on1 + LINE匿名相談窓口（ミニEAP）
- 価格モデル案: 施設月額5〜15万円 + セッション従量
- 訴求: 「看護師1人の離職コスト＞年間契約料」の置き換え提案（KHDの経営コンサル文脈にそのまま乗る）
- この段階で法人化を検討:
  - 形態: 合同会社 or 株式会社のJV
  - 出資比率案: **友人51% : KHD49%**（当事者ブランドの信頼性維持を優先）または50:50
  - IP整理: メソッド・発信＝友人帰属、システム・営業資産＝KHD帰属、共同開発物は契約で明記

## 5. リスクと手当て

| リスク | 手当て |
|--------|--------|
| 友人関係と金銭の混在 | Phase 0から**書面（覚書）**。無償支援でも範囲と期限を明文化 |

| 相談内容の守秘・個人情報 | LINE運用は同意取得・匿名化設計。B2B展開時は病院に個人が特定される情報を渡さない設計が商品価値そのもの |
| 属人性（友人が倒れたら止まる） | Phase 1でメソッドの教材化・グループ化を進め、労働集約から脱却 |
| 本人が「事業化」を望まない可能性 | その場合は撤退し、友人としての応援（発信ノウハウ提供）に留める |

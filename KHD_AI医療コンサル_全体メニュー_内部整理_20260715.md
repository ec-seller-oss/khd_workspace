# KHD AI医療コンサル ── 全体メニュー（内部整理用）

作成日: 2026-07-15
用途: 菊池さん自身が「AI医療コンサルとして何ができるか」を一覧で把握するための内部資料。
お客様提示用ではないため、社外配布時は`KHD_iryo_consul_sales.pptx` / `ai_iryo_consul_template.pptx` / `KHD_iryo_consul_talkscript.md`を都度カスタマイズして使うこと（情報量・トーンは相手に合わせる）。

---

## 1. 立ち位置・コンセプト

- **フック**：無料の「診療圏ミニ診断」。売り込まず、まずGIVE。
- **中核信条**：特定ツールを売らない中立の立場。「院の現実」と「既存の電カル/予約業者（高額・縦割り・自社製品売り）」のあいだの隙間に入る。
- **型（5ステップ）**：①無料診断 → ②刺さる1つを選ぶ → ③0円で小さく試す（AIで実物を作る）→ ④数値で測る → ⑤効いた所を拡大。
- **差別化**：
  - 宅建士・土地家屋調査士の資格 → 立地の数字だけでなく物件（再建築・境界・契約リスク）まで一気通貫で見れる。
  - 叔父の代からの医療テナント26年の実務を継承 → 開業から承継まで同じ顔で伴走。
  - AIで「提案書」でなく「動く実物」を0円で作って見せる（他社は資料止まり）。

## 2. 料金モデル（標準形・京橋モデル）

| 項目 | 内容 |
|---|---|
| 初期費用 | 0円 |
| 月額基本料 | 0円 |
| 成果報酬 | 削減額の40%（導入後6ヶ月間） |
| 7ヶ月目以降 | 月5万円の保守（月1回MTG込み） |
| 解約 | 効果0円なら費用0円。解約自由 |

※ 標準モデル試算（医師1・事務3名、月削減額 約24万円）：1〜6ヶ月は院に月14.4万円残る／7ヶ月目以降は月19万円残る／12ヶ月累計で院に約200万円。
※ 効果測定の基準は `業務委託契約書 第2条3(1)` に基づく「ベースライン確認書」（導入前後の入力・記載時間を院側に記入してもらう様式）で運用。他社事例の追加は要検討（顧客ごとに情報量の調整が必要なため保留中）。

## 3. ソリューション・メニュー（課題 → 打ち手）

| 院の課題 | 主な打ち手 | 具体的な内容 |
|---|---|---|
| ① 受付・問い合わせ・電話の負担 | 公式LINE | FAQ自動応答／Web問診／予約／リマインド・お知らせ／リッチメニュー設計（予約・Web問診・FAQの3タップ導線） |
| ② 電子カルテの入力・転記の負担 | Stream Deck・AI入力 | 定型文のワンタッチ入力、音声からの自動抽出・入力連携。効果測定はベースライン確認書で導入前後比較 |
| ③ 集患・認知が弱い | オウンドメディア・HP・SNS × AI | 発信の量産（KHD医療系YouTubeチャンネル、X投稿、HP）。医療×不動産×AIメディアとして信頼を積み上げる導線 |
| ④ 開業・分院・立地の判断 | 診療圏調査（AI量産） | 年齢別人口×受療率で需要を積み上げ→半径内競合で按分→「1日に何人来うるか」を高速に数値化。開業/承継（買収査定）/運営（競合監視）/撤退の4フェーズで継続的に使える |
| ⑤ 承継・M&A・補助金 | テナント承継コンサル＋IT導入補助金 | 居抜き・テナント承継の座組み、IT導入補助金の活用支援で導入コストを圧縮 |
| （周辺）物件×ニーズのマッチング | 薬局提携物件マッチングパイプライン | ウエルシア・日本調剤等の薬局提携物件と、先生の開業ニーズ（エリア×科目×坪）を月次で自動照合し、候補レポートを生成 |
| （周辺）オンライン診療の要否判断 | オンライン診療3社比較 | 導入要否を判断するための比較1枚資料（初診対応可否・コスト等） |
| （周辺）経営全体の目線合わせ | 医療経営グランドデザイン／事業構造比較ツール | 保険診療モデルの経営構造・倒産要因・逆算チェックをExcelで型化。院長との前提合わせ資料としても使える |

## 4. フェーズ別の打ち手（院のステージで主役が変わる）

| フェーズ | 主役ソリューション | 狙い |
|---|---|---|
| 開業・分院を考える院 | 診療圏調査（AI量産） | 「ここで本当に食えるか」を出店判断前に数値化 |
| 運営中で忙しい院 | 公式LINE／電カル入力効率化／オウンドメディア発信 | 受付・入力・発信の負担を課題に応じて個別に軽くする |
| 承継・出口を考える院 | 承継コンサル＋IT補助金 | テナント承継・居抜きの座組みと補助金活用まで伴走 |

## 5. 効果測定の指標（院と一緒に数字で確かめる）

- 受付の電話本数／日（自動応答導入後の減少）
- 入力・転記の時間（Web問診・AI入力導入後の短縮）
- 新患数／問い合わせ数（発信施策の効果）
- 定期受診の来院率（リマインド施策の効果）
- 月末の残業時間（業務平準化の効果）

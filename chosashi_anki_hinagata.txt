#　ひな形 隙間暗記カード（①目的②原因の型③主な添付）/ 2026-06-10
#　★全カード、アガルート「書式ひな形対策」の本文で必ず文言を裏取りすること★
#　用途=隙間での型の維持・抜けチェック。実際の作成練習(描く)は机で別にやる。
【ひな形】表題部所有者の更正登記：目的・原因・主な添付は？	目的=表題部所有者更正登記 / 原因=なし / 添付=更正を証する情報 ★要確認	第1編 表題部所有者
【ひな形】表題部所有者の表示変更登記：目的・添付は？	目的=表題部所有者の表示変更登記 / 添付=変更を証する情報(住所等) ★要確認	第1編 表題部所有者
【ひな形】土地表題登記：目的・原因・主な添付は？	目的=土地表題登記 / 原因=なし / 添付=地積測量図・所有者の住所証明情報 ★要確認	第2編 土地
【ひな形】土地地目変更登記：目的・原因・添付は？	目的=土地地目変更登記 / 原因=年月日地目変更 / 地積測量図は不要(代理権限証明書のみ) ★要確認	第2編 土地
【ひな形】土地地積変更登記：目的・原因・添付は？	目的=土地地積変更登記 / 原因=年月日地積変更(埋立等で物理的増減) / 添付=地積測量図 ★要確認	第2編 土地
【ひな形】土地地積更正登記：目的・原因・添付は？	目的=土地地積更正登記 / 原因=なし(錯誤) / 添付=地積測量図 ★要確認	第2編 土地
【ひな形】土地地目更正登記：目的・原因は？	目的=土地地目更正登記 / 原因=なし(当初から誤り・錯誤) ★要確認	第2編 土地
【ひな形】土地分筆登記：目的・原因・添付は？	目的=土地分筆登記 / 原因=なし / 添付=地積測量図(分筆後の各筆) ★要確認	第2編 土地
【ひな形】土地合筆登記：目的・原因・添付は？	目的=土地合筆登記 / 原因=なし / 地積測量図は不要。所有権登記ありなら登記識別情報・印鑑証明書 ★要確認	第2編 土地
【ひな形】土地滅失登記：目的・原因は？	目的=土地滅失登記 / 原因=年月日滅失(海没等) ★要確認	第2編 土地
【対比】地積「変更」と「更正」の違いは？	変更=埋立等で面積が物理的に増減 / 更正=当初から登記面積が錯誤。どちらも地積測量図 ★要確認	第2編 土地 対比
【対比】分筆と合筆、地積測量図の要否は？	分筆=必要 / 合筆=不要。ひっかけ頻出 ★要確認	第2編 土地 対比
【ひな形】建物表題登記：目的・原因・主な添付は？	目的=建物表題登記 / 原因=なし / 添付=建物図面・各階平面図・所有権証明情報・住所証明情報 ★要確認	第3編 建物
【ひな形】建物表題部変更(所在の変更/えい行移転)：原因は？	年月日所在変更(えい行移転の場合は「えい行移転」と記録) ★要確認	第3編 建物
【ひな形】建物表題部変更(種類・構造の変更)：原因は？	年月日種類変更 / 年月日構造変更(図面不要の場合あり) ★要確認	第3編 建物
【ひな形】建物表題部変更(床面積の変更=増築)：原因・添付は？	原因=年月日増築 / 添付=建物図面・各階平面図・増築部分の所有権証明情報 ★要確認	第3編 建物
【ひな形】建物表題部更正(床面積の更正)：原因・添付は？	原因=なし(錯誤) / 添付=建物図面・各階平面図 ★要確認	第3編 建物
【ひな形】建物分割登記：目的・原因・添付は？	目的=建物分割登記 / 原因=なし / 添付=建物図面・各階平面図 ★要確認	第3編 建物
【ひな形】建物分棟(表題部変更)：目的・課税は？	目的=建物表題部変更登記(分棟) / 登録免許税は課されない(報告的) ★要確認	第3編 建物
【ひな形】建物合併登記：目的・原因は？	目的=建物合併登記 / 原因=なし(同一性等の要件に注意) ★要確認	第3編 建物
【ひな形】建物滅失登記：目的・原因・添付は？	目的=建物滅失登記 / 原因=年月日取壊し(滅失) / 印鑑証明不要 ★要確認	第3編 建物
【対比】建物の「合体」と「合併」の違いは？	合体=物理的に1棟化(報告的・登録免許税なし・消滅承諾で全部消せる) / 合併=登記上1個にまとめる(要件厳格) ★弱点2位	第3編 建物 対比
【ひな形】建物合体による登記(表題登記なし＋表題登記あり)：申請期限は？	合体後1月以内に表題登記等を申請 ★要確認	第3編 建物 合体
【ひな形】区分建物表題登記(原始取得)：申請権者・主な添付は？	申請権者=原始取得者のみ / 添付=建物図面・各階平面図・規約証明・敷地権を証する情報等 ★要確認	第4編 区分建物
【ひな形】区分建物表題登記(48条2項代位)：誰が申請？	転得者が原始取得者に代位して申請(不登§47①・48②) ★要確認	第4編 区分建物
【ひな形】非区分建物から区分建物への変更：目的は？	建物区分登記/表題部変更(隔壁設置等で区分要件具備) ★要確認	第4編 区分建物
【ひな形】共用部分である旨の登記：主な添付・申請義務は？	添付=規約証明情報 / 申請義務=なし ★要確認	第4編 区分建物
【ひな形】区分建物合併登記(合併後も区分建物)：所有権証明は？	不要(合併は所有権証明書不要) ★要確認	第4編 区分建物
【対比】区分建物：原始取得者と転得者で申請方法は？	原始取得者=自ら表題登記 / 転得者=代位申請のみ(直接の表題登記申請は不可) ★要確認	第4編 区分建物 対比
【早見】登記原因「なし」になるのは？	分筆・合筆・地積更正・合併・(地目更正)。一方 地目変更=年月日地目変更/滅失=年月日滅失/増築=年月日増築 ★要確認	全編 原因の型

# KHD 仮想会社（Company）

菊池研太 / KHD の業務を回す仮想組織。秘書室を起点に、必要に応じて部署が増える。

## 現在の部署

| 部署 | 役割 |
|------|------|
| secretary（秘書室） | 受信・記録・タスク化・各専門家への割り振り。窓口。 |

## 三層記憶

- `inbox/` … 未処理の発話（生ログ）
- `tasks/open/` … 進行中タスク（1ファイル1タスク）
- `tasks/done/` … 完了タスク（移動・削除しない）
- `log/` … 完了・確定事項のアーカイブ
- `notifications/` … ユーザーへの通知

## 運用ルール

秘書室法（The Secretary Law）に従う。日本語で応答。受信即記録 → タスク化 → 完了で done へ。

## 連携

- 成果物・確定事項は Notion（SSoT: COMPANY KHD まとめ）へ
- 作業ログは Notion 作業ログDB / `khd-log` CLI
- 既存の KHD 本部制（01経営管理〜05物件調達）と整合させる
